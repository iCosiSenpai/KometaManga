Install / Usage

Jump into the project quickly

KometaManga works both for users who want to try it right away and for developers who want to build it locally.

01. Clone the repository

Grab the source code and prepare your local environment.

02. Install dependencies

Set up everything needed for build and development.

03. Run and customize

Start the project and adapt it to your reading workflow.

Terminal / Build 
$ git clone https://github.com/iCosiSenpai/KometaManga.git
$ cd KometaManga
$ npm install
$ npm run build
$ npm run start

# Or download the latest release directly
# and check the repository README for the full setup.
